# Mana Memory Detector

内存泄漏检测器

## 安装

在项目的 `pubspec.yaml` 中添加依赖：

```yaml
dependencies:
  mana_memory_detector: any
```

## 使用

```dart
import 'package:mana_memory_detector/mana_memory_detector.dart';

// 作为 Mana 插件使用（示例）
final plugin = ManaMemoryDetector();
```

## 说明

该插件为 Mana 平台的 Memory Detector 功能模块。
